# Vova_Vario
Arduino Para Variometer

HardWare
- Arduino Nano Atmega 328p
- BMP280 3.3v / (6 pins) / I2C 
- OLED / 0.96 / 128x64 /(4 pins) / I2C
- Buzzer
- Button

BMP:
vcc -> 3.3v, 
SCL -> A5, 
SCA -> A4, 
CSB -> x, 
SDD -> 3.3v, 

OLED:
vcc -> 3.3v, 
SCL -> A5, 
SCA -> A4, 

Buzzer:
09 (buzzer -), 
10 (buzzer +), 

Button -> 04
